Форматы

Native Banner

Блоки в контенте — лояльные к вашим посетителям.
- Самый популярный вид баннеров
✔ Настраиваются для каждого сайта – с учетом дизайна и вёрстки
✔ Используется проверка видимости
✔ Рефреш блока при продолжительном нахождении в зоне видимости

Top Sticky Banner

Горизонтальный баннер в верхней части экрана
- При скролле фиксируется на 2 секунды, обеспечивая максимальную видимость
✔ Настраивается для каждого сайта – с учетом дизайна и вёрстки
✔ Размер баннера адаптивный
✔ Время фиксации настраивается
✔ Количество показов пользователю настраивается

Sticky Sidebar

Вертикальный баннер для десктопа в боковой панели сайта.
- При скролле фиксируется в верхней части экрана
✔ Настраивается для каждого сайта – с учетом дизайна и вёрстки
✔ Размер баннера адаптивный
✔ Время фиксации настраивается
✔ Количество показов пользователю настраивается

Mobile Overlay

Горизонтальный баннер для мобильных устройств в нижней части экрана.
- Закрывается при клике на крестик
✔ Размер оболочки:
- 300×100px и 300×200px
✔ Размер баннера адаптивный
✔ Время закрытия настраивается
✔ Количество показов настраивается

Desktop Overlay

Горизонтальный баннер для десктопа. Появляется в нижней части экрана.
- Не перекрывает контент.
Закрывается при клике на крестик
✔ Размер оболочки адаптивный
✔ Размер баннера до 336×280px
✔ Время закрытия настраивается
✔ Количество показов настраивается

Multi Mobile Overlay

Два баннера для мобильных устройств в верхней и нижней части экрана.
- Верхний фиксируется на 2 секунды, нижний закрывается крестиком
✔ Размер оболочки:
- сверху и снизу 300×100px
✔ Размер баннеров адаптивный
✔ Время закрытия настраивается
✔ Количество показов настраивается
